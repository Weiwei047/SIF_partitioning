# SIF Partitioning

Weiwei Zhan

This repo shows the brief results of the SIF partitioning project, in which we show that neural networks (NN) informed by SIF observations can successfully be used to partition ecosystem carbon fluxes. 

More importantly, we find that NN can also retrieve the responses of the GPP–SIF relationship to environmental factors at the ecosystem scale. The figure below shows the NN-derived responses to atmospheric dryness.

![](Fig_10.jpg)

We have submitted the manuscript of this project, and the AGU presentation slides in `SIF_partitioning_AGU.pptx` show a summary of our methods and results.
