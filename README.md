# SIF Partitioning

Weiwei Zhan

This repo shows the brief results of the SIF partitioning project, in which we show that neural networks (NN) informed by SIF observations can successfully be used to partition ecosystem carbon fluxes and retrieve the GPP–SIF relationship at the ecosystem scale.

We have submitted the manuscript of this project, and the AGU presentation slides in `SIF_partitioning_AGU.pptx` show a summary of our methods and results.
